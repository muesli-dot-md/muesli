# Muesli documentation

The public documentation lives at **https://docs.muesli.md** — a Docusaurus site
maintained in the [muesli-dot-md/docs](https://github.com/muesli-dot-md/docs)
repository (getting started, guides, concepts, storage backends, agents/MCP,
self-hosting, and the CLI reference).

When you change user-facing behavior in this repo — commands, flags, env vars,
endpoints, UI flows — update the corresponding page there in the same breath.
